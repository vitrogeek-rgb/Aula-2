1° Abrir o VSCODE
2° No terminal digitar: cd desktop
3° No terminal desntro de desktop: mkdir projeto
4° No terminal: cd projeto
5° Dentro da pasta projeto: code .
6° No terminal da pasta projeto: touch index.html
7° No terminal: mkdir src
8° No terminal: cd src
9° No terminal: mkdir css
10° No terminal: mkdirr js 
11° No terminal cd css
12° No terminal: touch estilo.css
13° No terminal: cd ..
14° No terminal: cd js
15° No terminal: touch script.js
16° No terminal:cd ..
17° No terminal: cd .. até chegar na raiz do projeto
18° No terminal: git init
19° No terminal se estiver na Fiap: git config --global user.name "primeiro nome"
20° No terminal se estiver na Fiap: git config --global user.email email github
21° No terminal: git status
22° Se estiver vermelho: git add .
23° Se estiver verde: git commit -m "menssagem"
24° Entrar no Github em repositórios, criar um novo respositorio, colocar o nome e deixar público, clicar no botão criar repositório
25° No bloco que estiver as 3 linhas :
    1: linha- conecta seu gitbash com o seu github
    2: linha- muda o nome master para main
    3: linha- pega tudo que estiver no projero e sobe para o github
26° Caso você esteja em outro pc: no seu repositório no botão verde(code). copie o link
27° Abra o seu vscode no terminal cd desktop: git clone (cole o link), isso vai clonar o projeto para a máquina.
28° Se modificar o código:
    1: git status
    2: git add.
    3: git commit -m "menssagem"
    e para subir "git push"
